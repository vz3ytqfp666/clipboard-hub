# Clipboard Hub

A lightweight cloud clipboard with shared multi-device text sync, modern UI, security hardening, and clean Flask architecture.

For full documentation, see:
- `docs/REMOD.MD`
- `docs/API.MD`
- `docs/ARCHITECTURE.MD`

Run locally:
```
python run.py
```

Features:
- Multi-device shared clipboard
- Modern UI with dark/light themes
- SQLite persistence
- Secure API with validation & headers
- Modular backend structure
